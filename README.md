# Cloud Engineer Lab

This lab demonstrates hands-on cloud engineering, DevOps, automation, monitoring, and troubleshooting skills.

## Skills Practiced

- Python / Flask application development
- Docker containerization
- Git and GitHub workflow
- Jenkins CI/CD pipeline
- Prometheus monitoring
- Grafana dashboards
- Kubernetes with kind
- Helm chart deployment
- Terraform Infrastructure as Code
- AWS S3 static website hosting
- AWS EC2 web server deployment
- Security groups and SSH access
- Ansible automation and outage repair

## Lab Highlights

- Built and containerized a Python Flask app.
- Automated builds and container deployment with Jenkins.
- Added health and metrics endpoints.
- Monitored the app with Prometheus and Grafana.
- Deployed the app to Kubernetes using kubectl and Helm.
- Provisioned AWS S3 and EC2 infrastructure with Terraform.
- Used Ansible to troubleshoot and repair a stopped NGINX service on EC2.

## Troubleshooting Practiced

- Jenkins Docker socket access
- GitHub large file push error from Terraform provider cache
- AWS security group inbound rule validation
- SSH public key authentication failure
- EC2 instance type Free Tier issue
- Kubernetes CrashLoopBackOff
- Grafana to Prometheus container networking
